## 1. 誤差逆伝播法（Backpropagation）とは

ニューラルネットワークが「ミスから学ぶ」仕組み。

### 直感的な例え
目隠しでシュートを打つ場面を想像する：
1. シュートを打つ（**前向き伝播**）
2. 友人が「左に10cmズレた」と教える（**誤差計算**）
3. そのフィードバックをもとにフォームを調整（**誤差逆伝播**）
4. 入るまで繰り返す（**勾配降下法**）

### 全体のループ
```
入力データ → 前向き伝播 → 損失計算 → 誤差逆伝播 → 重み更新
    ↑___________________________________________________|
                     何千回も繰り返す
```

---

## 2. 連鎖律（Chain Rule）の役割

### なぜ必要か

ニューラルネットワークは**層を重ねた合成関数**：

```
L = f₃( f₂( f₁(x) ) )
```

「最も深い層の重み w₁ が、最終損失 L にどれだけ影響するか」を求めるために連鎖律が必要。

### 生活の例え

> 原油価格が上がる → 輸送費が上がる → 野菜の値段が上がる
>
> - 原油1円上昇 → 輸送費2円上昇
> - 輸送費1円上昇 → 野菜3円上昇
> - **∴ 原油1円上昇 → 野菜 2×3 = 6円上昇**

数式：
```
dL/dw₁ = dL/df₃ × df₃/df₂ × df₂/df₁ × df₁/dw₁
```

### 連鎖律がもたらす2つの問題

| 問題 | 発生条件 | 影響 | 対策 |
|------|----------|------|------|
| **勾配消失** | 各層の勾配 < 1 の連乗 | 深い層ほど勾配が 0 に近づき学習不能 | ReLU 活性化関数 |
| **勾配爆発** | 各層の勾配 > 1 の連乗 | 重み更新が発散、収束しない | 勾配クリッピング |

### 計算の再利用

誤差逆伝播は後ろから順に計算し、各層の勾配を**一度だけ計算して前の層に渡す**。
これにより、パラメータごとに個別に差分を取る方法と比べ計算量が**数百万分の1**になる。

## 3. 3層ネットワークの具体的な計算

### ネットワーク構成

```
入力層 (2ノード) → 隠れ層 (2ノード) → 出力層 (1ノード)
```

- 入力：x₁=1, x₂=0
- 目標：y=1
- 活性化関数：Sigmoid　σ(z) = 1/(1+e⁻ᶻ)
- 損失関数：MSE　L = ½(ŷ − y)²
- 学習率：η = 0.5

### 初期パラメータ

| 区間 | 重み | 値 | バイアス | 値 |
|------|------|----|----------|----|
| 入力→隠れ | w₁₁ | 0.5 | b₁ | 0.1 |
| 入力→隠れ | w₁₂ | 0.2 | b₂ | 0.1 |
| 入力→隠れ | w₂₁ | 0.3 | — | — |
| 入力→隠れ | w₂₂ | 0.8 | — | — |
| 隠れ→出力 | v₁  | 0.6 | b₃ | 0.1 |
| 隠れ→出力 | v₂  | 0.4 | — | — |

---

## 4. 前向き伝播（Forward Propagation）

### ステップ1：入力層 → 隠れ層

```
z₁ = w₁₁·x₁ + w₂₁·x₂ + b₁ = 0.5×1 + 0.3×0 + 0.1 = 0.6
z₂ = w₁₂·x₁ + w₂₂·x₂ + b₂ = 0.2×1 + 0.8×0 + 0.1 = 0.3

h₁ = σ(0.6) ≈ 0.6457
h₂ = σ(0.3) ≈ 0.5744
```

### ステップ2：隠れ層 → 出力層

```
z₃ = v₁·h₁ + v₂·h₂ + b₃
   = 0.6×0.6457 + 0.4×0.5744 + 0.1
   = 0.3874 + 0.2298 + 0.1
   = 0.7172

ŷ = σ(0.7172) ≈ 0.6718
```

### 損失計算

```
L = ½(0.6718 − 1)² ≈ 0.0539
```

---

## 5. 誤差逆伝播（Backward Propagation）

### ステップ1：損失の勾配

```
∂L/∂ŷ = ŷ − y = 0.6718 − 1 = −0.3282
```

### ステップ2：出力層の δ（誤差信号）

```
σ'(z₃) = ŷ·(1−ŷ) = 0.6718×0.3282 ≈ 0.2204

δ₃ = ∂L/∂ŷ × σ'(z₃) = −0.3282 × 0.2204 ≈ −0.0723
```

出力層の重み勾配：
```
∂L/∂v₁ = δ₃ × h₁ = −0.0723 × 0.6457 ≈ −0.0467
∂L/∂v₂ = δ₃ × h₂ = −0.0723 × 0.5744 ≈ −0.0415
∂L/∂b₃ = δ₃                           ≈ −0.0723
```

### ステップ3：隠れ層の δ（連鎖律の核心）

誤差を重みで伝播：
```
∂L/∂h₁ = δ₃ × v₁ = −0.0723 × 0.6 ≈ −0.0434
∂L/∂h₂ = δ₃ × v₂ = −0.0723 × 0.4 ≈ −0.0289
```

Sigmoid の導関数：
```
σ'(z₁) = h₁·(1−h₁) = 0.6457×0.3543 ≈ 0.2287
σ'(z₂) = h₂·(1−h₂) = 0.5744×0.4256 ≈ 0.2444
```

隠れ層の delta：
```
δ₁ = ∂L/∂h₁ × σ'(z₁) = −0.0434 × 0.2287 ≈ −0.0099
δ₂ = ∂L/∂h₂ × σ'(z₂) = −0.0289 × 0.2444 ≈ −0.0071
```

隠れ層の重み勾配：
```
∂L/∂w₁₁ = δ₁ × x₁ = −0.0099 × 1 ≈ −0.0099
∂L/∂w₁₂ = δ₂ × x₁ = −0.0071 × 1 ≈ −0.0071
∂L/∂w₂₁ = δ₁ × x₂ = −0.0099 × 0 =  0      ← x₂=0 のため更新なし
∂L/∂w₂₂ = δ₂ × x₂ = −0.0071 × 0 =  0      ← x₂=0 のため更新なし
```

---

## 6. 重みの更新

**更新式：新しい重み = 古い重み − η × 勾配**

| パラメータ | 旧値 | 勾配 | 新値 | 変化 |
|-----------|------|------|------|------|
| v₁  | 0.6000 | −0.0467 | **0.6234** | ↑ |
| v₂  | 0.4000 | −0.0415 | **0.4208** | ↑ |
| b₃  | 0.1000 | −0.0723 | **0.1362** | ↑ |
| w₁₁ | 0.5000 | −0.0099 | **0.5050** | ↑ |
| w₁₂ | 0.2000 | −0.0071 | **0.2036** | ↑ |
| w₂₁ | 0.3000 | 0       | **0.3000** | — |
| w₂₂ | 0.8000 | 0       | **0.8000** | — |

### 重要な観察

- 全勾配が**負** → 重みを増やすことで損失が減少する方向
- w₂₁, w₂₂ は x₂=0 のため更新されない（「貢献しない入力」は学習しない）
- 隠れ層の勾配は出力層より小さい → Sigmoid による**勾配消失の萌芽**
- この1イテレーション後、ŷ は 0.6718 から目標値 1.0 に少し近づく

---

## 7. 勾配の流れ（全体像）

```
L = 0.0539
    │
    │ ∂L/∂ŷ = −0.3282
    ▼
[出力層]  δ₃ = −0.0723  ──→  v₁, v₂, b₃ を更新
    │
    │ × v₁, v₂ で伝播
    ▼
[隠れ層]  δ₁=−0.0099, δ₂=−0.0071  ──→  w₁₁, w₁₂ を更新
    │
    │ × x₁=1, x₂=0
    ▼
[入力層]  x₂=0 → w₂₁, w₂₂ は更新されない
```

## 8. よくある疑問 Q&A

### Q1：∂L/∂ŷ = ŷ − y はなぜそうなるか

損失関数 L = ½(ŷ − y)² を ŷ で微分する：

```
∂L/∂ŷ = ½ × 2(ŷ − y) × 1 = ŷ − y
```

½ をかけていた理由はまさにここで **2 と打ち消すため**。

数直線で見ると：
```
         ŷ=0.6718      y=1
──────────────┬─────────┬──────→
              └────┬────┘
               差 = −0.3282
               （ŷ は y より小さい）
```

勾配の符号と意味：

| 状況 | ŷ | y | 勾配 | 意味 |
|------|---|---|------|------|
| 予測が小さすぎ | 0.67 | 1.0 | **−0.33**（負） | ŷ を増やせ |
| 予測がピッタリ | 1.00 | 1.0 | **0**（ゼロ） | 変更不要 |
| 予測が大きすぎ | 1.30 | 1.0 | **+0.30**（正） | ŷ を減らせ |

勾配の**符号**が修正方向、**大きさ**が修正量を教える。

---

### Q2：δ₃ = ∂L/∂ŷ × σ'(z₃) はなぜ2つを掛けるか

重み v₁ の更新には ∂L/∂v₁ が必要だが、v₁ は直接 L に触れていない：

```
v₁ → z₃ → ŷ → L
```

連鎖律で展開すると：

```
∂L/∂v₁ = ∂L/∂ŷ × ∂ŷ/∂z₃ × ∂z₃/∂v₁
           ↑          ↑          ↑
        −0.3282     0.2204      h₁
        (ズレ)   (Sigmoidの傾き) (v₁の貢献)
```

最初の2つをまとめたものが δ₃：

```
δ₃ = ∂L/∂ŷ × σ'(z₃)
   = −0.3282 × 0.2204
   = −0.0723
```

| 要素 | 値 | 役割 |
|------|----|------|
| ∂L/∂ŷ = −0.3282 | 予測のズレ | どちらに・どれだけ間違えたか |
| σ'(z₃) = 0.2204 | Sigmoid の傾き | z₃ の変化が ŷ にどう伝わるか |
| δ₃ = −0.0723 | 合成した誤差信号 | z₃ を増やせば損失が減る |

絶対値が 0.3282 → 0.0723 に小さくなったのは Sigmoid（0.2204）が信号を**圧縮**したから。

δ₃ があれば各重みの勾配を1行で書ける：
```
∂L/∂v₁ = δ₃ × h₁ = −0.0723 × 0.6457 = −0.0467
∂L/∂v₂ = δ₃ × h₂ = −0.0723 × 0.5744 = −0.0415
∂L/∂b₃ = δ₃ × 1  = −0.0723
```

---

### Q3：∂z₃/∂v₁ = h₁ になるのはなぜか

z₃ の定義：

```
z₃ = v₁·h₁ + v₂·h₂ + b₃
```

これを v₁ で偏微分すると、v₁ を含まない項（v₂·h₂ と b₃）は消える：

```
∂z₃/∂v₁ = h₁
```

`y = 3x` を x で微分すると `3` になるのと同じ構造。

**h₁ は「v₁ がどれだけ z₃ に貢献したか」の係数**。前の層の出力がそのまま勾配のスケールになる。

∂L/∂v₁ の完全な展開：

```
∂L/∂v₁ = ∂L/∂ŷ  ×  ∂ŷ/∂z₃  ×  ∂z₃/∂v₁
        = −0.3282 ×  0.2204  ×  0.6457
        =   δ₃               ×  h₁
        = −0.0723             ×  0.6457
        = −0.0467
```

---

### Q4：隠れ層の逆伝播（∂L/∂h₁ → δ₁ → ∂L/∂w₁₁）を詳しく

#### 全体の経路

```
w₁₁ → z₁ → h₁ → z₃ → ŷ → L
```

連鎖律で展開：
```
∂L/∂w₁₁ = ∂L/∂h₁ × ∂h₁/∂z₁ × ∂z₁/∂w₁₁
```

これを3段階で計算する。

---

#### 段階①：誤差を重みで伝播 ∂L/∂h₁

z₃ = v₁·h₁ + v₂·h₂ + b₃ を h₁ で偏微分すると `∂z₃/∂h₁ = v₁`。

```
∂L/∂h₁ = ∂L/∂z₃ × ∂z₃/∂h₁
        =  δ₃    ×   v₁
        = −0.0723 × 0.6
        = −0.0434
```

ポイント：**前向きでは `h₁ × v₁ → z₃`、逆向きでは `δ₃ × v₁ → h₁の誤差`**。重みが逆方向にも使われる。

---

#### 段階②：Sigmoid の導関数 σ'(z₁)

```
∂h₁/∂z₁ = σ'(z₁) = h₁·(1−h₁)
          = 0.6457 × 0.3543
          ≈ 0.2287
```

前向き伝播で計算済みの h₁ をそのまま使い回せる。

Sigmoid の傾きの性質：
- 出力が 0.5 付近 → 傾き最大（≈0.25）
- 出力が 0 や 1 付近 → 傾きほぼ 0 ← **勾配消失の原因**

---

#### 段階③：隠れ層の delta δ₁

```
δ₁ = ∂L/∂h₁ × σ'(z₁)
   = −0.0434 × 0.2287
   ≈ −0.0099
```

出力層との比較：

| | 出力層 | 隠れ層 |
|--|--------|--------|
| 式 | δ₃ = ∂L/∂ŷ × σ'(z₃) | δ₁ = ∂L/∂h₁ × σ'(z₁) |
| 前の誤差 | −0.3282 | −0.0434 |
| Sigmoid傾き | 0.2204 | 0.2287 |
| delta | −0.0723 | −0.0099 |

層を越えるたびに誤差信号が**小さくなっている**。

---

#### 段階④：重み勾配 ∂L/∂w₁₁

z₁ = w₁₁·x₁ + w₂₁·x₂ + b₁ を w₁₁ で偏微分すると `∂z₁/∂w₁₁ = x₁`。

```
∂L/∂w₁₁ = δ₁ × x₁ = −0.0099 × 1 = −0.0099
```

---

#### w₂₁, w₂₂ がゼロになる理由

```
∂L/∂w₂₁ = δ₁ × x₂ = −0.0099 × 0 = 0
∂L/∂w₂₂ = δ₂ × x₂ = −0.0071 × 0 = 0
```

z₁ を w₂₁ で偏微分すると `∂z₁/∂w₂₁ = x₂ = 0`。
x₂=0 なので w₂₁ の z₁ への貢献がゼロ → 責任もゼロ → 勾配もゼロ → 更新されない。

---

#### 連鎖律の全体展開

```
∂L/∂w₁₁ = ∂L/∂ŷ  ×  ∂ŷ/∂z₃  ×  ∂z₃/∂h₁  ×  ∂h₁/∂z₁  ×  ∂z₁/∂w₁₁
         = −0.3282 ×  0.2204  ×   0.6     ×   0.2287  ×    1
         =    δ₃            ×   v₁      ×  σ'(z₁)  ×   x₁
         = −0.0099
```

5つの項が連鎖してひとつの勾配になっている。

---

## 9. キーポイント まとめ

| テーマ | ポイント |
|--------|---------|
| 誤差逆伝播の本質 | 最終誤差を各パラメータに「逆向きに責任分配」する |
| 連鎖律の役割 | 合成関数の微分を「各層の影響の積」に分解する |
| 勾配消失 | Sigmoid の連乗で深層の勾配が消える → ReLU で対策 |
| 勾配爆発 | 勾配が連乗で発散する → 勾配クリッピングで対策 |
| ゼロ入力問題 | x=0 の入力につながる重みは更新されない |
| 計算効率 | 後ろから順に計算し勾配を再利用 → 計算量を劇的に削減 |
